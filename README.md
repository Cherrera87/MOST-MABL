# MOST-MABL

Aplicación interactiva para visualizar estabilidad en la capa límite marina usando Monin–Obukhov:
- Ejes: **u\*** (m s⁻¹) vs **H** (W m⁻²)
- Banda **CASI NEUTRO** basada en |z/L| ≤ ζ₀
- Regímenes **INESTABLE/ESTABLE**
- Apuntes (LaTeX).

## Requisitos
- MATLAB (recomendado R2018b o superior).
- No requiere toolboxes especiales.

## Ejecutar
1. Descarga o clona el repositorio.
2. En MATLAB, navega a la carpeta del repo.
3. Ejecuta:

```matlab
MO_marina_interactivo
